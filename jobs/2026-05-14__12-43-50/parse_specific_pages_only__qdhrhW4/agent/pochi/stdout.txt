# Parse Only Specific Pages of a PDF with LlamaParse (`page_ranges.target_pages`) ## Background LlamaCloud is LlamaIndex's managed RAG-as-a-Service platform. Its agentic document parser, LlamaParse, turns visually rich PDFs into clean markdown. By default LlamaParse parses every page of a document, which is wasteful when you only need a subset. For partial parses, the `parsing.parse` API accepts a top-level `page_ranges` argument with a `target_pages` field that takes a comma-separated string of **1-based** page numbers and ranges (e.g. `"1,3,7-12"`). Pages outside of that selection are skipped entirely and their content does not appear in the result. In this task you must build a small parsing pipeline that takes a pre-existing 4-page PDF (`catalog.pdf`) and parses only **pages 1 and 3** using the `llama-cloud` Python SDK, then writes the combined markdown text of those two pages to `selected_pages.md`. ## Requirements Write a Python script `parse_selected.py` that uses the `llama-cloud` SDK (`from llama_cloud import LlamaCloud`) to: 1. Upload the local PDF `catalog.pdf` to LlamaCloud with `purpose="parse"`. 2. Submit a parse job with all of the following options: - `tier="agentic"` - `version="latest"` - `page_ranges={"target_pages": "1,3"}` (LlamaParse must only parse pages 1 and 3) - `expand=["markdown"]` 3. Concatenate the `markdown` field of every page returned by the parser (`result.markdown.pages`) in order, using a single newline as the separator between pages. 4. Write that concatenated markdown to `selected_pages.md` in the same directory (UTF-8 encoded). The script must authenticate via the `LLAMA_CLOUD_API_KEY` environment variable (the SDK reads it automatically when `LlamaCloud()` is instantiated with no arguments). Do NOT hardcode an API key. ## Implementation Guide 1. Change into the project directory `/home/user/catalog_task`. 2. Create `parse_selected.py` with logic similar to: ```python from llama_cloud import LlamaCloud client = LlamaCloud() # reads LLAMA_CLOUD_API_KEY from the environment uploaded = client.files.create(file="./catalog.pdf", purpose="parse") result = client.parsing.parse( file_id=uploaded.id, tier="agentic", version="latest", page_ranges={"target_pages": "1,3"}, expand=["markdown"], ) parts = [(p.markdown or "") for p in result.markdown.pages] combined = "\n".join(parts) with open("selected_pages.md", "w", encoding="utf-8") as f: f.write(combined) ``` 3. Execute the script with `python3 parse_selected.py` from `/home/user/catalog_task`. The script will block until LlamaCloud finishes parsing and then writes the markdown file. 4.
